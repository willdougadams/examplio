# README

Welcome to [Examplio](https://examplio.netlify.app/)!

> **Prerequisites**
Examplio is built using [RedwoodJS](https://redwoodjs.com/docs/introduction)

clone from github, then
```
yarn install
yarn rw dev
```

Navigate to [http://localhost:8910](http://localhost:8910)
